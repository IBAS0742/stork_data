## 数据更新

- 更新 ```龙虎榜``` 数据

```cmd
:: 启动数据库服务
simpledb.exe --config=storkYZ.json --mode=web
node 龙虎榜数据.js
```

- 更新 ```日k```

```cmd
:: 启动服务 
simpledb.exe --config=stork.json --mode=web
node 抓取所有股票日 k
```

- 更新 ```分时```

```cmd
node getStorkData.js
node insertAllStorkFSData2db.js
storkSql/merge.bat
```
